# Myfinance::Rails

API integration with MyFinance (www.myfinance.com.br)

# API Documentation

https://app.myfinance.com.br/docs/api 

## Installation

Add this line to your application's Gemfile:

    gem 'myfinance-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install myfinance-rails
